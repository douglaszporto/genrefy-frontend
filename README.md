# Genrefy Front-end

Front-end construido em React JS para consumir a API do Genrefy-Backend

## Dependencias

 - Node v8.9.4
 - NPM 6.5.0

## Instalação
```
git clone https://github.com/douglaszporto/genrefy-frontend.git
cd genrefy-frontend
npm install
```

## Inicialização
Após ter a aplicação back-end rodando em http://localhost:3000 :
```
npm start
```
